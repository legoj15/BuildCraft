Microsoft/Mojang have recently changed how Minecraft versions work. It went from `main.major.minor` (e.g. 1.2.5, 1.12.2, 1.18.0 1.21.11, etc.) to year.major.minor (e.g. 26.1, 26.1.1). This can cause some confusion, because "Minecraft version 26.1" Doesn't match up with the previously expected versioning scheme. So, as a quick reference, the following example versions are listed in release order (first is older, last is newer):

1.2.5
1.12.2
1.18.0
1.21.11
26.1
26.1.1

Currently NeoForge documentation only has up to 1.21.11
