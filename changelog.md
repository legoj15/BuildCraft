###### Changes since 2026.1.0:
- Added support for Minecraft 26.2
- The paintbrush can now recolour glazed terracotta, candles, and beds
- Searing fluids now emit steam particles in the world (config option available)
- Wrenches from other mods (anything tagged c:tools/wrench) now can wrench BuildCraft blocks. In addition, the BuildCraft wrench should now be able to wrench other mod's blocks. Thank you, [MapperTV](https://github.com/MapperTV)!
- Fixed the DISPLAY_RF option so that it now actually displays machine power readouts in RF/FE instead of MJ. Thank you, [MapperTV](https://github.com/MapperTV)!
- The Laser's hitbox now matches its model (a base plate plus the emitter), so you can click and walk between closely-packed lasers instead of bumping an invisible full cube. Thank you, [MapperTV](https://github.com/MapperTV)!
- Added a "Laser Targeting Behavior" config option. The default LOS Cone keeps the modern 8.0.x behaviour (receivers must sit in a 6-block cone in front of the laser with a clear line of sight); Box restores the legacy 7.x reach, letting lasers power any table within 5 blocks to the front or sides, even through walls. Thank you, [MapperTV](https://github.com/MapperTV)!
