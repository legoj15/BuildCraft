###### Changes since 2026.1.0:
- Added support for Minecraft 26.2
- The paintbrush can now recolour glazed terracotta, candles, and beds
- Searing fluids now emit steam particles in the world (config option available)
- Wrenches from other mods (anything tagged c:tools/wrench) now can wrench BuildCraft blocks. In addition, the BuildCraft wrench should now be able to wrench other mod's blocks. Thank you for the suggestion, [MapperTV](https://github.com/MapperTV)!
- Fixed the DISPLAY_RF option so that it now actually displays machine power readouts in RF/FE instead of MJ. Thank you for the report, [MapperTV](https://github.com/MapperTV)!
- The Laser's hitbox now matches its model (a base plate plus the emitter), so you can click and walk between closely-packed lasers instead of bumping an invisible full cube. Thank you, [MapperTV](https://github.com/MapperTV)!
- Added a "Laser Targeting Behavior" config option to allow for switching between 8.0.x (1.11/1.12) and 7.1.x (1.7.10) laser handling.  And one last thank you to [MapperTV](https://github.com/MapperTV) for bringing this up.
- Fixed BuildCraft's facade scanner flooding the log with errors when other mods register custom block models on Minecraft 1.21.1. Thank you for the report, [RickWeek](https://github.com/RickWeek)!
