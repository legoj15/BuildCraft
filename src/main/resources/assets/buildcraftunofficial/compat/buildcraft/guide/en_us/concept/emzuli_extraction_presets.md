<lore>
The Emzuli Transport Pipe gives you multiple extraction filters; you pick one in a gate for it to use that filter.
</lore>
<no_lore>
Extraction Presets are gate actions provided on any gate placed on an Emzuli Transport Pipe.
</no_lore>

<chapter name="Requirements"/>
When the action is activated, the gate allows the colour of the selected Extraction Preset to be extracted by the Emzuli Pipe.
Without an extraction preset set by the gate, the Emzuli Pipe will be unable to extract any items.
