The Gate Copier will copy the data from a gate to allow it to be pasted onto other gates.
</no_lore>
<new_page/>
<chapter name="Information"/>
Interact with a gate to copy its current configuration. The tool will visually change and notify you that the information has been copied. Pasting is just as easy, interact with the Gate Copier tool in hand on whatever gates you wish to have the same configuration. To clear the copied data, crouch and interact with the gate copier in hand, allowing you to copy a different configuration.
