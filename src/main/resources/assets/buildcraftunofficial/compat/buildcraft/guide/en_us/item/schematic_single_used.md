Single Block Schematics are blueprints for a single block. It will capture all of the information for that specific block for it to be placed again by hand, or used in a Replacer.
</no_lore>
<chapter name="Information"/>
With an empty schematic, interact with any block, and the block's information will be written to it. This can be used to place identical copies of that block by hand if you have enough resources in your inventory, or it can be used to replace existing blocks in a blueprint by using a Replacer with two difference single block schematics.
<link to="buildcraftunofficial:block/replacer"/>
To wipe a single block schematic to use it for something else, crouch and interact with it.
If you're looking for a way to copy entire structures exactly, or just their general shape, check out Blueprints and Templates.
