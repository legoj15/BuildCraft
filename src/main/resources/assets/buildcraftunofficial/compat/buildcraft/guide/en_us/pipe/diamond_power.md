<chapter name="Pipe Mechanics"/>
The Diamond Kinesis Pipe carries up to 256 MJ/t by default - the highest of any Kinesis Pipe - and, like the Iron Kinesis Pipe, acts as a power limiter. Right-click it with a wrench to cycle the cap through seven steps, from its full rate down to nothing, so even the largest power line can be metered precisely.

The current limit is shown on the pipe and confirmed in an on-screen message each time you change it. Power is fed into the network by a Wooden Kinesis Pipe.
<link to="buildcraftunofficial:pipe/wood_power"/>

<chapter name="Gate Control"/>
A gate placed on the pipe can set the same limit automatically with the Set Power Limit actions, swapping the cap in response to any trigger.
