<chapter name="Pipe Mechanics"/>
The Diamond FE Pipe carries up to 2,560 FE/t by default - the highest of any FE Pipe - and, like the Iron FE Pipe, acts as a limiter. Right-click it with a wrench to cycle the cap through seven steps, from its full rate down to nothing, so even the largest line can be metered precisely.

The current limit is shown on the pipe and confirmed in an on-screen message each time you change it. Forge Energy is fed into the network by a Wooden FE Pipe.
<link to="buildcraftunofficial:pipe/wood_rf"/>

<chapter name="Gate Control"/>
A gate placed on the pipe can set the same limit automatically with the Set Power Limit actions, swapping the cap in response to any trigger.
<link to="buildcraft:set_power_limit"/>
